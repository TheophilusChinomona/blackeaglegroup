## Image Paths Used in Code

### COT Event Pages
- Carousel images: `/COT/img/carousel-1.jpg`, `/COT/img/carousel-2.jpg`
- Welcome section: `/COT/img/quote.jpg`
- Sponsor logos: `/COT/img/H3C.png`, `/COT/img/TakeNote.png`, etc.
- PDF icon: `/COT/img/flag.png`
- PDF document: `/COT/img/ProgramBusiness.pdf`

### CSIR Event Pages
- Logo: `/csir/images/Logo(1).png`
- PDF icon: `/csir/images/flag.png`
- PDF document: `/csir/images/INFO_FINAL_23June.pdf`
- Sponsor logos: `/csir/Logos/H3C.jpg`, `/csir/Logos/Amscor2.jpg`, etc.
- Sponsor tier badges: `/csir/Logos/Diamond.png`, `/csir/Logos/Gold.png`

### CASSI Gallery
- Gallery images: `/cassi/img/projects/...` or `/cassi/img/...`
