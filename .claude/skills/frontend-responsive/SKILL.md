# Frontend Responsive

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend responsive design.

## When to use this skill

- When implementing responsive layouts with breakpoints (sm, md, lg, xl)
- When using mobile-first CSS approach (base styles for mobile, enhance for larger screens)
- When creating fluid containers with max-width and percentage-based widths
- When implementing responsive grid or flexbox layouts
- When using relative units (rem, em) instead of fixed pixels
- When optimizing images for different screen sizes (srcset, responsive images)
- When implementing touch-friendly design with 44x44px minimum touch targets
- When testing layouts across device sizes (375px mobile, 768px tablet, 1024px+ desktop)
- When hiding or showing content based on viewport size
- When ensuring readable typography at all screen sizes
- When creating responsive navigation (hamburger menus, collapsible navigation)
- When implementing responsive spacing and padding
- When optimizing performance for mobile devices
- When ensuring content priority on smaller screens
