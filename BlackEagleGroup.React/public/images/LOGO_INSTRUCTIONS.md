# Logo Placement Instructions

## Current Logo Location
The website logo is located at:
- **Path**: `BlackEagleGroup.React/public/images/Website logo.png`
- **Used in**: Header component (`src/components/common/Header.jsx`)

## To Update the Logo

1. **Place your new logo file** in the `BlackEagleGroup.React/public/images/` folder
2. **Name it**: `Website logo.png` (or update the reference in Header.jsx if using a different name)
3. **Recommended format**: PNG with transparent background (if needed) or PNG/JPG
4. **Recommended size**: Height of 50-80px for optimal display (the current CSS sets height to 50px)

## Logo Specifications
Based on the logo description:
- Logo should feature: "BLACK EAGLE GROUP HOLDINGS Pty Ltd & SRM"
- Eagle emblem with shield and crown
- Dark green text on black background (or appropriate colors for website header)

## Current Usage
The logo is referenced in:
- `src/components/common/Header.jsx` (line 49)
- Path: `/images/Website logo.png`

## After Adding the Logo
1. Replace the existing `Website logo.png` file, OR
2. Update the path in `Header.jsx` if using a different filename
3. Restart the dev server if needed
4. The logo will automatically appear in the website header
