# Animated Buttons Modal Spec

## Overview
Implement animated buttons from `@animate-ui/components-buttons-button` in the contact form modal to enhance user interaction.

## Requirements
- Install the necessary shadcn/animated-ui component.
- Identify the modal contact form component (likely `EventContactModal.jsx` or similar).
- Replace standard buttons with the animated variants.
- Ensure styling matches the Black Eagle Group brand (colors, fonts).

## Component Identification
- Search for "modal" and "contact" to find the correct file.
- Look for Submit/Cancel buttons within that file.
