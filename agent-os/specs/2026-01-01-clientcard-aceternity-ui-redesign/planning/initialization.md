## Context

Aceternity UI is a React component library featuring modern, animated UI components with effects like:
- **Card Components:** 3D Card Effect, Evervault Card, Wobble Card, Card Spotlight, Focus Cards, Glare Card, Direction Aware Hover, Card Hover Effect
- **Effects:** Background gradients, spotlight effects, moving borders, hover animations
- **Animations:** Framer Motion-based animations, micro-interactions

The current ClientCard component displays client information with:
- Logo area
- Company name (serif typography)
- Location & industry metadata
- Call and Reference action buttons
- Brand guide styling: 4px green top border, hover lift effect
